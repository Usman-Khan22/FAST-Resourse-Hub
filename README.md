# FAST Resource Hub

FAST Resource Hub is a simple front-end web project built using HTML, CSS,
and JavaScript. This project was created for practice and learning purposes
as part of my first semester coursework.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Simple and clean user interface
- Organized sections for study resources
- Basic interactivity using JavaScript

## Purpose
This project was created to practice front-end web development
and improve understanding of HTML, CSS, and JavaScript.

## Notes
- This is a front-end only project.
- The project may be updated with new features as I continue learning.
